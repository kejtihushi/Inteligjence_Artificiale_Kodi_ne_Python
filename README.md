Kodi në Python 

Testimi i Algoritmeve të Klasifikimit në Python (Google Colab)

Ky repository përmban kodin Python për testimin e algoritmeve të klasifikimit mbi dataset-et Iris, Wine, Breast Cancer dhe Digits.  

Algoritmet e përdorura janë:  
- Decision Tree  
- Random Forest  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  
- Artificial Neural Network (MLP)  

Përshkrimi i Dataset-eve

1. Iris Dataset – 150 rreshta, 4 veçori, 3 klasa.  
2. Wine Dataset – 178 rreshta, 13 veçori, 3 klasa.  
3. Breast Cancer Dataset – 569 rreshta, 30 veçori, 2 klasa.  
4. Digits Dataset (mini MNIST) – 1797 shembuj, 8x8 pixel, 10 klasa.

Random seeds janë fiksuar për të siguruar riprodhueshmërinë e rezultateve.

Deep Learning – CNN për Fashion MNIST
Ky repository përmban kodin Python për testimin e një modeli Convolutional Neural Network (CNN) mbi dataset-in Fashion MNIST (70,000 imazhe, 28x28 pixels, 10 klasa). Modeli përbëhet nga dy shtresa Conv2D + MaxPooling, Flatten, Dense dhe softmax. Performanca matet me Accuracy, Precision, Recall dhe F1-Score.
